Esercizio 2 - CSS INTRODUZIONE

1) INLINE CSS

Lo stile inline può essere usato per applicare uno stile unico per un singolo elemento.

Per usare gli stili in linea, aggiungi l'attributo style all'elemento pertinente.

L'attributo style può contenere qualsiasi proprietà CSS.

Esempio:

<p style="color:blue; margin-left:30px;">Questo è un testo con uno stile.</p>

2) INTERNAL CSS

Un foglio di stile interno può essere usato se una singola pagina ha uno stile unico.

Gli stili interni sono definiti all'interno dell'elemento <style>, all'interno della sezione <head> di una pagina HTML:

Esempio:

<head>
    <meta charset="UTF-8">
    <title>Esercizio 2</title>
    <style>
        body {background-color: rgb(124, 235, 124)}
        h1.titolo{text-align:center}
        h2 {color:red}
    </style>
</head>

3) EXTERNAL CSS

Con un foglio di stile esterno, puoi cambiare l'aspetto di un intero sito cambiando un solo file.

Ogni pagina deve includere un riferimento al file del foglio di stile esterno all'interno dell'elemento <link>.

L'elemento <link> entra nella sezione <head>.

Esempio:

<head>
    <link rel="stylesheet" type="text/css" href.="mystyle.css"(senza il punto)>
</head>

Contenuto mystyle.css:

body {
        background-color: lightblue;
    }
    
h1 {
        color: navy;
        margin-left: 20px;
    }

4) A cosa servono le seguenti proprietà dei css?

4a° - color

La proprietà color assegna un colore ad un elemento.

4b° - font-family

La proprietà font-family assegna un font ad un elemento.

4c° - font-size

La proprietà font-size assegna una grandezza ad un elemento.

4d° - border

La proprietà border assegna lo stile, la larghezza e il colore del bordo di un elemento.

4e° - padding

La proprietà padding viene utilizzata per generare spazio attorno al contenuto di un elemento, all'interno di qualsiasi bordo definito.

4f° - margin

La proprietà margin viene utilizzata per creare lo spazio attorno agli elementi, al di fuori dei confini definiti.
