ESERCIZIO 1 - CSS ADVANCED

1) Cosa significa CSS?

CSS sta per Cascading Style Sheets.

2) A cosa servono?

CSS descrive come gli elementi HTML devono essere visualizzati su schermo, carta o su altri supporti.

3) Qual è il “grosso problema” che risolvono i CSS rispetto alla formattazione html?

HTML non ha mai avuto lo scopo di contenere tag per la formattazione di una pagina web!

HTML è stato creato per descrivere il contenuto di una pagina Web, ad esempio:

<h1> Questa è un'intestazione </h1>    
<p> Questo è un paragrafo. </p>

Quando tag come <font> e attributi di colore sono stati aggiunti alle specifiche HTML 3.2, è iniziato un incubo per gli sviluppatori web.

Lo sviluppo di grandi siti Web, in cui i caratteri e le informazioni sul colore sono stati aggiunti a ogni singola pagina, è diventato un processo lungo e costoso.

Per risolvere questo problema, il World Wide Web Consortium (W3C) ha creato CSS.

Il CSS ha rimosso la formattazione dello stile dalla pagina HTML!

5) Cosa si intende per...

a° Element Selector.

L'Element Selector seleziona tutti gli elementi con il nome dell'elemento specificato.

b° IdSelector.

L'IdSelector disegna l'elemento con l'id specificato

c° Class Selector.

Il Class Selector seleziona gli elementi con un attributo di classe specifico. Per Selezionare elementi con una classe specifica, scrivi un carattere punto (.), seguito dal nome della classe.

6) Fai tre esempi...

Esempio Element Selector:

<head>
    <style>
        p{background-color:yellow;}
    <style>
<head>

Esso seleziona e abbina tutti gli elementi di <p>.


Esempio IdSelector:

<head>
    <style>
        #firstname{background-color: yellow;}
    <style>
<head>

Esso modella l'elemento con l'id = "#firstname".


Esempio Class Selector:

<head>
     <style>
        .intro {background-color: yellow;}
     <style>
<head>

Seleziona e affina tutti gli elementi con class = "intro".

7) Che differenza c'è tra clase e id?

Classe...

Le classi servono per definire una tipologia di elementi, cioè per attribuire uno scopo e/o una presentazione 
ad un sottoinsieme di elementi con caratteristiche e funzionalità comuni di una pagina HTML.

ID...

Gli id servono per 
definire un elemento unico in una pagina, con un unico e preciso scopo, nella maggior parte dei casi tale scopo 
è determinare una sezione in una pagina HTML.

8) Prova il seguente codice:

Codice...

<head>
    <style>
        p.center {
        text-align: center;
        color: red;
        }
    </style>
</head>
    <body>
        <h1 class="center">This heading will not be affected</h1>
        <p class="center">This paragraph will be red and center-aligned.</p>
        <p>This heading will not be affected.</p>
    </body>

Risultato del codice...

This heading will not be affected

This paragraph will be red and center-aligned.

This heading will not be affected.


a° Quale paragrafo viene formattato?

Viene formattato il paragrafo che ha class= "center".

b° Perchè non viene formattato l'h1?

L'h1 non viene formattato perchè abbiamo messo la classe center ha un tag <p>.

c° Perchè non viene formattato il secondo paragrafo?

Perchè nel secondo paragrafo non è stata inserita la classe "center".

d° Spiega la notazione p.center.

La notazione p.center è un "class selector" ed esso seleziona tutti gli elementi che hanno come class="center".

9) Prova il seguente codice...

Codice...

<!DOCTYPE html>
<html>
    <head>
        <style>
            h1, h2, p {
                text-align: center;
                color: red;
            }
        </style>
    </head>
        <body>
            <h1>Hello World!</h1>
            <h2>Smaller heading!</h2>
            <p>This is a paragraph.</p>
        </body>
    </html>

Risultato del codice...

Hello World!

Smaller heading!

This is a paragraph.

a° Per quale ragione lo stile viene applicato a tutti i tag?

Perchè a nessun tag è stata inserita nessuna classe particolare e quindi a tutti i tag di quel tipo viene applicato quello stile.

b° Come si raggruppano più selettori con la stessa formattazione?

Mettendo la virgola tra i vari selettori.
